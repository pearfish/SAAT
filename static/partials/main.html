Angular!

this app demonstrates some simple CRUD, three main pages:

  • view promos
  • new promo
  • edit promo

enumeration of routes, their templates, and their controllers is handled in routeProvider in app.config
each page is pulled from a template HTML file, which contain may contain just plain HTML or minor angular logic
the controllers outline simple rules associated with a view, and scope management
services/factories are used for more heavy lifting, in this example they're used for handing http requests
